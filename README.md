# 🎓 «Кім боламын?» — Кәсіби бағдар Telegram боты

7-8 сынып оқушыларына болашақ мамандық таңдауға көмектесетін Telegram-бот.

## 🎯 Мәселе

Қазақстанда 7-8 сыныпта профиль таңдау басталады, бірақ кәсіптік бағдар беру формальды.
Оқушылар қандай мамандықтар сұраныста екенін білмейді.

## 💡 Шешім

Бот сұрақтар қояды (қызығушылық, күшті жақтар, сүйікті пәндер),
жауаптарды талдап, **ТОП-5 мамандық** ұсынады.
Әр мамандыққа:
- 📋 Сипаттама
- 📚 Қажет ҰБТ пәндері
- 🏫 ҚР ЖОО-лары
- 💰 Жалақы деңгейі
- 📈 Сұраныс деңгейі

## ⚙️ Технология

- **Python** + **aiogram 3** (Telegram Bot API)
- **JSON** мәліметтер қоры (25 мамандық, 17 ЖОО)
- **FSM** (Finite State Machine) — сауалнама күйлерін басқару

## 📋 Оқу бағдарламасына сәйкестігі

| Тақырып | Қолданылуы |
|---------|-----------|
| Шартты операторлар | Тег скорларын салыстыру, мамандық таңдау логикасы |
| Мәліметтерді өңдеу мен талдау | JSON-нан деректер оқу, жауаптарды талдау |
| Тізімдер мен сөздіктер | Тегтер, скорлар, мамандықтар |
| Файлмен жұмыс | JSON файлдарды оқу |
| Циклдар | Сұрақтар тізімін айналу, мамандықтарды сұрыптау |

## 🚀 Іске қосу

```bash
# 1. Тәуелділіктерді орнату
cd bot
pip install -r requirements.txt

# 2. .env файлын жасау
echo "BOT_TOKEN=your_telegram_bot_token" > .env

# 3. Ботты іске қосу
python main.py
```

## 📁 Жоба құрылымы

```
bot/
├── main.py              # Бот іске қосу
├── config.py            # Конфигурация
├── data/                # JSON мәліметтер қоры
│   ├── questions.json   # 15 сұрақ (3 блок)
│   ├── professions.json # 25 мамандық
│   └── universities.json # 17 ЖОО
├── handlers/            # Telegram хендлерлер
├── keyboards/           # Inline + Reply клавиатуралар
├── services/            # Талдау алгоритмі
└── models/              # FSM күйлері, сессия моделі
```
