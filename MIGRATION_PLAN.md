# План перехода проекта: Математика PISA → Физика «Нанотехнология негіздері»

## Тақырып / Тема

**«ОРТА МЕКТЕПТЕРДЕГІ ФИЗИКАНЫ ОҚЫТУДА "НАНОТЕХНОЛОГИЯ НЕГІЗДЕРІ" БӨЛІМІНЕ ЭЛЕКТИВТІ КУРС БАҒДАРЛАМАСЫН ӘЗІРЛЕУ»**

Разработка программы элективного курса по разделу «Основы нанотехнологий» в преподавании физики в средних школах.

---

## 1. Обзор текущего проекта

**Math PISA Bot** — Telegram Mini App для изучения школьной математики по модели PISA на казахском языке.

| Параметр | Значение |
|----------|----------|
| Стек | React 18 + Vite 5 + TailwindCSS, FastAPI + SQLAlchemy, aiogram 3, Groq AI |
| Сервисы | frontend, backend, bot, admin-panel |
| Язык интерфейса | Казахский |
| Кол-во исходных файлов | ~100 |
| Строк кода | ~11 000 |
| Предметная область | 4 PISA-домена математики |

**Целевой проект: Physics Nanotechnology Bot** — тот же стек и архитектура, но предметная область меняется на элективный курс «Нанотехнология негіздері» для учащихся средних школ.

## 2. Новая структура предметной области

### 2.1 Темы (топики) вместо PISA-доменов

| Старый topic_id | Старое название (каз.) | Новый topic_id | Новое название (каз.) | Описание |
|-----------------|------------------------|----------------|----------------------|----------|
| `quantity` | Сан және шама | `atomic_structure` | Атом құрылысы | Атомдар, молекулалар, нанобөлшектер құрылысы |
| `change_and_relationships` | Өзгерістер мен тәуелділіктер | `quantum_basics` | Кванттық физика негіздері | Толқындық-бөлшектік дуализм, энергия деңгейлері |
| `space_and_shape` | Кеңістік пен пішін | `nanomaterials` | Наноматериалдар | Наноқұрылымдар, фуллерендер, нанотүтіктер, графен |
| `uncertainty_and_data` | Анықсыздық пен деректер | `nano_applications` | Нанотехнология қолданыстары | Медицина, электроника, энергетика саласындағы қолданыстар |

### 2.2 Деңгейлер (уровни сложности)

Сохраняется шкала 1–6, но адаптируется под физику:

| Деңгей | Сипаттама (каз.) | Описание |
|--------|------------------|----------|
| 1 | Негізгі түсініктер | Базовые понятия нанотехнологий |
| 2 | Қарапайым есептеулер | Простые расчёты (размер атома, масштаб) |
| 3 | Құрылым талдау | Анализ атомной/молекулярной структуры |
| 4 | Қасиеттерді болжау | Прогнозирование свойств наноматериалов |
| 5 | Синтез және модельдеу | Синтез и моделирование нанопроцессов |
| 6 | Зерттеу деңгейі | Исследовательские задачи |

### 2.3 Контент для каждой темы

Каждая тема включает 3 подтемы, в каждой — 3-4 формулы LaTeX.

**Тема 1: Атом құрылысы**
- Бор моделі (модель Бора): $E_n = -13.6/n^2$ эВ
- Де Бройль толқын ұзындығы: $\lambda = h / (mv)$
- Гейзенберг анықсыздық принципі: $\Delta x \cdot \Delta p \geq \hbar/2$

**Тема 2: Кванттық физика негіздері**
- Шрёдингер теңдеуі, энергия кванттау
- Фотоэлектрлік эффект: $E = h\nu - A$
- Туннельдік эффект

**Тема 3: Наноматериалдар**
- Наночастиктер беткі аудан/көлем қатынасы: $S/V = 6/d$
- Фуллерен, графен, нанотүтіктер құрылысы
- Механикалық, оптикалық, электрлік қасиеттер

**Тема 4: Нанотехнология қолданыстары**
- Медицинадағы наноботтар
- Нано-электроника, кванттық нүктелер
- Энергетика: күн батареялары, катализаторлар
